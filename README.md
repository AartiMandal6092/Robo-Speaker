# Robo-Speaker

**Robo Speaker** is a simple Python program that lets your computer **speak any text you type**.  

It's a fun way to learn Python and text-to-speech functionality on Windows.

---

## Features

- Converts any typed text into **spoken audio**  
- Handles empty input gracefully  
- Easy to use: just run the Python file  

---

## Requirements

- Windows OS  
- Python 3.x  

No external Python modules are required since it uses **Windows PowerShell and System.Speech**.

---

# How to Use

1. Download or clone the `robo_speaker.py` file.  
2. Open Terminal or Command Prompt in the folder containing the file.  
3. Run the program:
